## `impressions init`

Bootstrap a new Impressions project in the current directory.

Usage:

```bash
impressions init
```

This will create:

.
├── impressions.toml
├── tasks/
│   └── example.yaml
└── reports/

If a file already exists, you'll be prompted before it is overwritten. Use `--force` or `-f` to overwrite without prompting.
